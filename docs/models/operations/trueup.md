# Trueup

Controls whether future trueup invoices are billed or removed. Default behavior is AS_IS if not specified.


## Values

| Name                | Value               |
| ------------------- | ------------------- |
| `TrueupRemoveLower` | remove              |
| `TrueupAsIsLower`   | as_is               |
| `TrueupRemoveUpper` | REMOVE              |
| `TrueupAsIsUpper`   | AS_IS               |
